## Ajude no desenvolvimento
Instale o [Node.js](https://nodejs.org).

Clone o repositório.\
`git clone https://github.com/elder-tome/movie-tv.git`

Crie uma conta no [The Movie Database API](https://www.themoviedb.org/login).\
Solicite uma [API KEY](https://developers.themoviedb.org/3/getting-started/introduction).\
Na pasta do projeto crie um arquivo **.env** com a variável ambiente **REACT_APP_TOKEN** e atribua a ela o valor da sua API KEY.

Instale as dependências do projeto.\
`npm i`

Inicie a aplicação.\
`npm start`
